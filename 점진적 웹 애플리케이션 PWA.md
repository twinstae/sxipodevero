어떤 [[실행 환경 Running Environment|실행환경]]에서도 동일한 [[웹 표준 Web Standard]]으로 돌아가는 [[웹 Web]]의 장점과, [[네이티브 앱 Native App]]의 장점을 결합한 방식.

웹 사이트처럼 편리하게 접속해서, 원하면 앱처럼 설치한 뒤에 파일 시스템, 카메라, GPS 같은 기능을 제공하는 [[API]]를 이용해 풍부한 기능을 제공할 수 있다.
